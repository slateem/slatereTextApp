Text Analytics App

App Description: This is a 3 part Text Analytics App. The app offers a way to get sample content, translate text from any language to English, and to count the number of times each word is used. All of these features work independently of each other.

Get Sample Content (Uses Teamate's scraper services)

Instructions: If you want sample content, enter a word and click "Click Here for Sample Output". Sample Content will be printed to the screen. If you already have content, you can skip this step

Translate Text (Uses my transformation service)

Instructions: If you need to translate your content to English, enter content (in any language) into the text box and press "Click Here for Translated Text". If you do not have to translate content, you can skip this step
